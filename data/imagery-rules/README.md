## 使用纪律

1. 四柱、十神、藏干、实际交运时间和确定干支关系优先。
2. 正式规则均包含成立条件、削弱因素、现实验证点和禁止越级结论。
3. 扫描资料中未逐页人工核读的正文只进入 `research-pending.json`，不会打包进正式 `IMAGERY_RULES`。
4. 案例断语不得直接升格为通则。
5. 死亡、牢狱、重大疾病、流产、寿命等高风险断语不进入自动用户答案。
